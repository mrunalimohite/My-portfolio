# Personal Portfolio Website

A modern and responsive portfolio website built using React.js, showcasing my skills, projects, experience, and contact information.

## 🚀 Live Demo

https://your-portfolio-link.netlify.app

## 📌 Features

- Responsive Design
- Modern UI/UX
- Project Showcase
- Skills Section
- Work Experience Timeline
- Contact Form
- Mobile-Friendly Navigation
- Smooth Scrolling

## 🛠️ Built With

- React.js
- CSS3
- React Icons
- JavaScript (ES6+)
- Vite

## Installation

Clone the repository:

```bash
git clone https://github.com/mrunalimohite/My-portfolio.git
```

Navigate to the project folder:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
